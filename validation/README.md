# validation/

Valideringsrapporter fra ChatGPT (kvalitetskontroll-rollen) og andre QA-agenter.

## Innhold

- `calendar-validation.md` — Sjekk av calendar.json mot Ark T
- `geometry-validation.md` — Sjekk av kjernekonstanter og posisjoner
- `marker-validation.md` — Sjekk av historiske markører (koordinater, kilder)
- `solbane-validation.md` — Sjekk av sol-bane-arkT.json mot Excel
- `url-validation.md` — URL-sjekk av siterte kilder (hallusinerings-vern)

Hver valideringsrapport skal inneholde:
- Hva ble sjekket
- Hvilken metode
- Hva ble funnet
- Eventuelle avvik
- Anbefalinger
