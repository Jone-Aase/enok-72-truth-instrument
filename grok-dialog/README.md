# grok-dialog/

Arkiv over dialog mellom Jone-Aase og Grok Heavy. Sikrer kontinuitet når Perplexity-agenten resettes.

## Navnekonvensjon

`YYYY-MM-DD-kort-emne.md`

Eksempler:
- `2026-05-27-pipeline-godkjent.md`
- `2026-05-28-endring-01-glod-puls.md`
- `2026-05-29-endring-01-deploy-feedback.md`

## Hva som skal logges

- Beslutningsdialog (hvorfor en endring ble valgt før en annen)
- Spørsmål Jone-Aase stilte og Grok sine svar
- Alternativer som ble vurdert og forkastet
- Verbatim formuleringer som senere kan misstolkes
- Småjusteringer som ikke kom med i commit-meldinger
